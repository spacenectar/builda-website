# Modal

The Modal component is a simple container component which can display anything in a modal window (i.e. hovers over the screen).

<Tip type="note">
  The 'blackout' and 'wrapper' sections cannot be restyled. The custom styling
  will only apply to the 'content' sections.
</Tip>

<Canvas withSource="open">
  <Story name="Default">
    {(args) => {
      const [isOpen, setIsOpen] = useState(false);
      return (
        <>
          <Button onClick={() => setIsOpen(true)}>Open Modal</Button>
          <Modal isOpen={isOpen} setIsOpen={setIsOpen} {...args} />
        </>
      );
    }}
  </Story>
</Canvas>

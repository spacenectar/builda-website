# Table

The Table component is used to display a table of data. In order to use all of the features of the Table component, you will need to use the `Table.Head`, `Table.Body`, `Table.Row`, and `Table.Cell` components.

<Tip>
  It is strongly recommended that you use the{' '}
  <a href="?path=/docs/organism-tablegenerator--default-story">TableFactory</a>{' '}
  component to generate the table, instead of using the Table component
  directly. This will ensure that the table is styled correctly and is able to
  use all available features, such as sorting and filtering.
</Tip>

<Tip type="danger">
  If you are manually creating the table, you should add the `contextHeading`
  prop to each instance of `TableCell`, otherwise it will not work correctly at
  smaller layouts. These examples have been built without the contextHeading
  props, try it at a smaller layout to see what happens.
</Tip>
